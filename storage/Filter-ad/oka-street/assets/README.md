# 素材替換說明 (Assets)

目前 `frames/` 與 `dino/` 內都是**佔位用的 SVG**，可直接替換成你的正式圖檔。

## 兩種素材

| 資料夾 | 用途 | 規格建議 |
|--------|------|----------|
| `frames/` | 拍照時疊在相機畫面上的「相框」 | **中央必須透明**、直式 9:16（如 1080×1920）、PNG |
| `dino/` | 解鎖頁卡片 / 地圖上的恐龍縮圖 | 正方形（如 512×512）、可透明、PNG |

## 怎麼換

**最簡單**：把新圖命名成跟現有檔一樣，覆蓋掉即可（副檔名也用 `.svg`）。

**換成 PNG**：放入 PNG 後，到 [`../js/config.js`](../js/config.js) 把對應的路徑副檔名改掉，例如：

```js
frame: 'assets/frames/trex.png',
thumb: 'assets/dino/trex.png',
```

## 地區 ↔ 恐龍 對應（檔名）

| 地區 | 恐龍 | frame / dino 檔名 |
|------|------|-------------------|
| 台中（科博館，預設）| 暴龍 | `trex` |
| 台北 | 三角龍 | `triceratops` |
| 宜蘭 | 劍龍 | `stegosaurus` |
| 台南 | 翼龍 | `pteranodon` |
| 台東 | 迅猛龍 | `velociraptor` |

> 相框的座標、地區中心點都在 [`../js/config.js`](../js/config.js)，要新增/調整地區改那一個檔即可。
